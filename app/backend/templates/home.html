Description

This is a application to predict the edibility of a mushroom by a number of features
Notably, in this version the features used are bruising, odor, and gill-size.
This application uses a feature selected random forest classifier to classify the features.
an image classification application is in the works, feel free to post a photo to help in training!

Using this application

This application has the following endpoints:

  • info
  • permissions
  • prediction(get)
  • prediction(post)
  • and upload(post)
You can use this application headless in the terminal, or on this webpage.

Go to /info to see how to fillout the prediction form
Go to /prediction/[anything] to download the fillout form
Post the fillout form to /prediction/[filename] to process the form
Go to /permissions to check what info will be stored and to revoke/accept form
