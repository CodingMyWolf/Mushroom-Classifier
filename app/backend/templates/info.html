Mushroom Classifier Application

By: Matteo Vidali

Below are the instructions to using this appliction


Make a get request to the prediction endpoint to get a fillout form.
The template of this form may varyover time, but for now it has three featurs.
More endpoints may be added in the future, with varying levels of accuracy.
You can parse the following list of options for how to fill out the form.
You will use the one letter code which best describes the feature of the
mushroom to be classified:
(The emboldened features are the current ones in the fillout form)

options: cap-shape: ["bell=b, conical=c, convex=x," "flat=f, knobbed=k,sunken=s"]
cap-surface: [fibrous=f, grooves=g, scaly=y, smooth=s]
